## Introduction

An Android application using Java, XML, Volley Framework,... used to note daily tasks that need to be done.

## Technology
The application is built with: 
* Java, XML, Android Studio,..
  * Back - end
    * Rest API
    * SharedPreferences
    * Volley Framework
    * JSON 
  * Front - end
    * Recycler View
    * Interface
    * Layout
    * Gradient Color
    * Navigation
    * ..
## Features
* Create account, login or logout
* Save account
* Create/Update/Delete task to do
* Follow task by date
* Mark the task as completed
